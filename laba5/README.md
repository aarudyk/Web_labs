# Лабораторна робота 5 — Task Manager (DOM, події, API)

Інтерактивний Todo App для **Артем Рудик**, група 136.

**Гілка:** `lab-05`  
**Репозиторій:** [github.com/aarudyk/Web_labs](https://github.com/aarudyk/Web_labs)

## Можливості

- Завантаження завдань з [JSONPlaceholder](https://jsonplaceholder.typicode.com)
- Додавання, видалення, перемикання статусу (POST / PATCH / DELETE)
- Фільтри: Всі / Активні / Виконані
- Пошук з debounce (300 мс)
- Делегування подій на списку завдань
- Клавіатура: Enter — додати, Escape — очистити поле
- `Promise.all` для завантаження todos + user
- Індикатор завантаження та повідомлення про помилки

## Файли

- `index.html` — розмітка застосунку
- `styles.css` — стилізація
- `script.js` — логіка (vanilla JavaScript)

## Запуск

Відкрийте `index.html` у браузері (потрібен інтернет для API).
