# Лабораторна робота 2 — CSS-стилізація

Стилізоване HTML-портфоліо Артема Рудика (група 136).

**Гілка:** `lab-02`  
**Репозиторій:** [github.com/aarudyk/Web_labs](https://github.com/aarudyk/Web_labs)

## Що реалізовано

- Зовнішній файл `styles.css` підключений до всіх сторінок
- Селектори: елемент, клас, id, комбіновані, атрибут, псевдокласи, псевдоелементи
- Box Model: margin, padding, border, border-radius, `box-sizing: border-box`
- Фіксований хедер та кнопка «Нагору»
- `position: relative` + `absolute` (бейдж на аватарі)
- Google Fonts (Roboto, Poppins), кольори HEX / RGB / HSL, розміри в `rem`
- Стилізація навігації, таблиць, форми, посилань (LVHA + focus)
- `overflow: hidden` та `overflow-y: auto`

## Запуск

Відкрийте `index.html` у браузері.
