# Лабораторна робота 4 — JavaScript

Практичні задачі з основ JavaScript для студента **Артем Рудик**, група 136.

**Гілка:** `lab-04`  
**Репозиторій:** [github.com/aarudyk/Web_labs](https://github.com/aarudyk/Web_labs)

## Файли

- `index.html` — сторінка з виводом результатів
- `script.js` — реалізація 7 завдань

## Завдання

1. Змінні та типи даних (`const`, `let`, `typeof`, перетворення, template literals, `==` vs `===`)
2. Умови та логіка (`getGrade`, `getSeasonUA`, тернарний оператор)
3. Масиви (`push`, `pop`, `splice`, `find`, `filter`, `reduce`)
4. Функції (declaration, expression, arrow, closure, default/rest, деструктуризація)
5. Об'єкти (методи, ітерація, spread, optional chaining)
6. Ланцюжки методів масивів (`filter` → `map` → `reduce`, `sort`)
7. Рядки (`capitalize`, `countWords`, `truncate`, `isValidEmail`)

## Запуск

1. Відкрийте `index.html` у браузері
2. Перегляньте результати на сторінці
3. Відкрийте консоль (F12) для детального виводу `console.log`
