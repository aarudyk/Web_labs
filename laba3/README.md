# Лабораторна робота 3 — Адаптивна верстка

Адаптивне CSS-портфоліо Артема Рудика (група 136).

**Гілка:** `lab-03`  
**Репозиторій:** [github.com/aarudyk/Web_labs](https://github.com/aarudyk/Web_labs)

## Реалізовано

- CSS Grid макет (`grid-template-areas`: header, sidebar, content, footer)
- Flexbox навігація з CSS-only гамбургер-меню
- Сітка проєктів: `repeat(auto-fit, minmax(280px, 1fr))`
- Mobile First: брейкпоінти 768px та 1024px
- CSS Custom Properties (`:root`)
- Переходи (кнопки, картки, посилання) та анімація `fadeIn`
- Гнучка типографіка через `clamp()`
- Адаптивні зображення

## Сторінки

- `index.html` — головна
- `about.html` — про мене
- `projects.html` — проєкти
- `contacts.html` — контакти

## Запуск

Відкрийте `index.html` у браузері. Змініть ширину вікна або увімкніть режим мобільного пристрою в DevTools.
