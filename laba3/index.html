Мої ключові цілі

  1. Опанувати HTML, CSS та JavaScript
  2. Вивчити сучасний фреймворк (React або Vue)

Адаптивна верстка дозволяє одному сайту зручно працювати на телефоні, планшеті та десктопі. Mobile First означає, що базові стилі пишуться для мобільних пристроїв, а потім розширюються для ширших екранів.
